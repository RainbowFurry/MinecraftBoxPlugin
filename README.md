# 📦 Box

A modern and customizable Minecraft box system featuring private worlds, upgrades, friends, and random teleportation.

---

## ✨ Features

- 📦 Personal box worlds
- 👥 Friend system
- 🌍 Visit other players' boxes
- 🏷️ Rename your box
- ⬆️ Upgrade your box
- 🎲 Random Teleport (RTP)
- ⚡ Fast and lightweight

---

# 📖 Commands

| Command | Description |
|---------|-------------|
| `/box` | Opens your personal box menu. |
| `/box help` | Displays all available commands. |
| `/box rename <name>` | Renames your box. |
| `/box visit <player>` | Visits another player's box. |
| `/box friend add <player>` | Sends a friend invitation. |
| `/box friend remove <player>` | Removes a friend. |
| `/box friend join <player>` | Joins a friend's box (if permitted). |
| `/upgrade` | Opens the upgrade menu. |
| `/rtp <world\|nether\|end>` | Randomly teleports you within the selected dimension. |

---

# 🔑 Permissions

| Permission | Description | Default |
|------------|-------------|---------|
| `box.command.box` | Allows using `/box`. | `true` |
| `box.command.upgrade` | Allows using `/upgrade`. | `true` |
| `box.command.rtp` | Allows using `/rtp`. | `true` |
| `box.admin` | Grants access to administrative features. | `op` |

> Adjust the permission nodes to match your plugin if necessary.

# 🚀 Installation

1. Download the latest release.
2. Place the plugin into your server's `plugins/` folder.
3. Restart the server.
4. Configure the plugin to your liking.
5. Enjoy!
